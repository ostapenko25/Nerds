# Nerds
Layouts for web studio "Nerds". Static, optimize for 900px screens. HTML, CSS, pure JavaScript.
Include 2 pages:
1. Main page. Including modal windows on oure Javascript - "Feedback". Slider in promo-section on the Pure CSS.
2. Online-store.
